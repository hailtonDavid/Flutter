Visão Computacional para Reconhecimento de Padrões Dermatoglificos:
Identificação de crianças com perfil atlético de alto rendimento, ou
crianças com possíveis disturbios de aprendizagem.

É comum encontrar alunos com distúrbios de aprendizado. Entretanto, sua identificação no meio de milhares de alunos através de um diagnóstico preciso nem sempre é possível, em detrimento, talvez, da limitação do número de profissionais habilitados para executar testes que possam fazê-lo de forma concisa e segura. Neste trabalho propõe-se o uso de imagens digitais dos dermatoglifos dos dedos das mãos, com a finalidade de se identificar padrões dermatoglíficos que indiquem possíveis distúrbios de aprendizado através de um teste simples e não invasivo.
